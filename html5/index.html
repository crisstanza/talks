HTML5: O que é?

  • HTML5 é o guarda-chuva acima de tudo que há de novo na web
  • Resolve antigos problemas de versões anteriores
  • Poe em prática o conceito de plugin-free
  • Não está concluído

O que mudou

  • Novas tags (footer, header, article, section, picture, etc)
  • CSS3 (animações, transições, etc)
  • Audio e video (captura e processamento)
  • Formulários
  • Armazenamento local
  • Localizacao geográfica
  • Websockets
  • Design responsivo
  • Processamento de primitivas gráficas (SVG e Canvas)

Canvas

  • Do inglês tela (ou seja, uma área para desenho livre ponto a ponto)
  • Inicialmente desenvolvido pela Apple
  • Manipulaçao via script
  • Permite criar retangulos, circunferências e formas geométricas mais complexas
  • Integração total com HTML e CSS
  • Necessita browser moderno

Criando Canvas 

 <canvas id="canvas" width="200" height="100"></canvas>
// obtendo o elemento canvas
var canvas = document.querySelector("canvas"); 

// obtendo o contexto, nesse caso 2d
var ctx = canvas.getContext("2d");

// setando a cor do elemento para vermelho
ctx.fillStyle = "red";

// desenhando um retangulo com 30 de largura e 20 de altura
ctx.fillRect(0, 0, 30, 20);
