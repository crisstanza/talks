HTML5: O que e?

  • HTML5 tudo que ha de novo
  • Resolve antigos problemas de versoes anteriores
  • Poe em pratica o conceito de plugin-free
  • Nao esta concluido

O que mudou

  • Novas tags (article, section, picture, etc)
  • Audio e video (captura e processamento)
  • Formularios
  • Armazenamento local
  • Localizacao geografica
  • Websockets
  • Design responsivo
  • Processamento de primitivas graficas (Canvas)

Canvas

  • Do ingles tela (ou seja, uma area para desenho livre ponto a ponto)
  • Inicialmente desenvolvido pela Apple
  • Permite criar retangulos, circunferencias e formas geometricas mais complexas
  • Integracao total com HTML e CSS
  • Necessita browser moderno

Criando Canvas 

 <canvas id="canvas" width="200" height="100"></canvas>
// obtendo o elemento canvas
var canvas = canvas.querySelector("canvas"); 

// obtendo o contexto, nesse caso 2d
var ctx = canvas.getContext("2d");

// setando a cor do elemento para vermelho
ctx.fillStyle = "red";

// desenhando um retangulo com 30 de largura e 20 de altura
ctx.fillRect(0, 0, 30, 20);
